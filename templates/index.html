How to Subscribe

📱 iOS/macOS (Apple Calendar)

  1. Open Calendar app
  2. File → New Calendar Subscription (Mac) or Settings → Accounts → Add Account → Other → Add Subscribed Calendar (iOS)
  3. Paste the feed URL above
  4. Set refresh frequency (recommended: hourly)

📧 Google Calendar

  1. Open Google Calendar on web
  2. Click "+" next to "Other calendars"
  3. Select "From URL"
  4. Paste the feed URL above

🖥️ Outlook

  1. Open Outlook
  2. File → Account Settings → Internet Calendars
  3. Click "New" and paste the feed URL
